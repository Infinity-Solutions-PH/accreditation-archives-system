ACCREDITOR ACCESS CREDENTIALS
    => The admin will create temporary account for accreditor and send it to their Email

VIRTUAL ACCREDITATION ROOM
    => The admin can create a virtual drive for each accreditation
    => Virtual drive means a folder that contains only files related or can be accessed by the accreditor.
    => Example: Accreditation of BSIT Program Level II
        => There are already set of folders per area
        => The files inside each folder are shared by designated accreditation task force member of the program

ROLE CREATION (ROLE APPROVAL BASED)
    => Program Accreditation Taskforce, IDO STAFF or ADMIN (Pending) => Admin role will approved
    => Program Accreditation Taskforce (restrict per college) => College Accreditation Officer approved


ACCESS LEVEL:
    1. Admin (Only 1)
    2. IDO STAFF (Many)
        - Create or assign College Accreditation Officer per college

    3. College Accreditation Officer (Specific college) (1)
        - Share the link, download the file
        - Create or assign Program Accreditation Taskforce

    4. Program Accreditation Taskforce (Specific college and program) (Many)
        - Upload and manage files

    5. Accreditor (Temporary account only) (Specific college and program) (Many)
